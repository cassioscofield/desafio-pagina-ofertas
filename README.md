# Desafio Página Ofertas

----------------------------------------------------------------------

***SETUP***

$ *git clone https://github.com/cassioscofield/desafio-pagina-ofertas.git*

$ *cd desafio-pagina-ofertas*

$ *npm install*

$ *npm start*

$ *npm test*
